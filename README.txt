SCHOOLS WEB APP
-------

Web Edu is an education theme with an elegant look and feel, designed to meet the needs of Schools, Colleges, Institutes, eLearning Centers and Universities.  It’s provided with an nice responsive layer slider. This theme is very easy to customize asper you requirments.

 
Features :
--------
-Open the xampp control panel
-Start both the apache and mysql server
-Create a data base and atable both of them called client
-In the table add the id,Fname,Lname,gender,status, email,phone,class,subject,message  
-Click on the index.html file and start using the website
